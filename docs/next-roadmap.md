# 量化学习下一步路线

本路线对应课程 9.5。完成 `learn-quant-with-codex` 后，下一步不应该马上追求复杂模型，而是继续巩固数据、验证、报告和工程习惯。

## 0. 保留当前项目习惯

- 每个研究问题都先写清边界：教育用途，不构成投资建议。
- 每个策略都先说明假设和失效场景。
- 每个回测都保存数据范围、成本假设、signal / position 规则和指标局限。
- 每个结论都避免把历史结果写成未来收益承诺。

## 1. 第 1 个月：统计和概率基础

目标是能解释收益率分布、均值、方差、相关性和抽样误差。

建议产物：

- 用 `pandas` 生成收益率分布摘要。
- 比较均值、中位数、标准差和极端值。
- 写一页笔记说明“样本均值不等于未来期望收益”。

## 2. 第 2 个月：时间序列入门

目标是理解金融数据的时间顺序、滞后特征和非平稳性。

建议产物：

- 对一个价格序列计算 rolling mean、rolling volatility 和 rolling drawdown。
- 比较不同市场阶段的年度表现。
- 写一页笔记说明为什么时间序列不能随机打乱。

## 3. 第 3 个月：因子研究入门

目标是从单策略规则进入更系统的横截面研究。

建议产物：

- 用本地样例数据构造一个简单动量分数。
- 按分数排序，比较 top / bottom 组合的历史表现。
- 写一页笔记说明 selection bias、survivorship bias 和多重测试风险。

## 4. 第 4 个月：组合构建

目标是理解权重、风险暴露、再平衡和 benchmark。

建议产物：

- 比较等权、动量轮动和 benchmark。
- 记录权重漂移和再平衡成本。
- 写一页笔记说明“买很多股票不自动等于分散化”。

## 5. 第 5 个月：更真实的回测工程

目标是知道向量化回测和事件驱动回测的区别。

建议产物：

- 把当前 pandas 回测器的限制列成清单。
- 研究订单、成交、滑点、现金、持仓和交易日志的数据结构。
- 写一页笔记说明 paper trading 和 live trading 的差异。

## 推荐扩展方向

| 方向 | 先补什么 | 小项目 |
| --- | --- | --- |
| 因子研究 | 统计、分组收益、IC | 做一个动量因子研究报告 |
| 组合优化 | 协方差、约束、风险预算 | 比较等权和简单风险平价 |
| 回测系统 | 事件、订单、撮合、现金 | 写一个极小订单日志模拟器 |
| 数据工程 | 数据版本、质量报告、缓存 | 给样例数据加版本化质量检查 |
| 研究表达 | 可复现实验、限制章节 | 把 Capstone 改成可发布研究笔记 |

## 最低验收

- 能选择一个下一步方向。
- 能写出 3 个要补的知识点。
- 能定义一个 2 周内可完成的小项目。
- 继续保留“历史结果不代表未来收益”的报告边界。
