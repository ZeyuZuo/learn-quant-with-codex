# 课程调研与设计

## 调研目的

这个项目的课程不是给有金融背景的人看的，而是给金融小白看的。

课程设计需要先帮学习者建立投资常识、股票常识和风险意识，再进入量化投资。否则学习者会很容易把量化理解成“找公式赚钱”，而不是一套围绕市场、数据、假设、验证和风险控制的研究流程。

## 调研来源

- Investor.gov / SEC: 投资入门内容强调投资目标、风险、投资产品、股票市场如何运转、费用和防欺诈。
- FINRA Investing Basics: 入门顺序包括金融基础、投资目标、风险、买卖、费用、资产配置、分散化、绩效评估和波动率。
- MIT OpenCourseWare Finance Theory I: 金融课程会先讲金融系统和现值，再讲股票、债券、衍生品、风险收益、组合理论、CAPM 和有效市场。
- MIT OpenCourseWare Investments: 投资课程强调金融理论、股票市场实证、固定收益、市场效率、主动投资、行为金融，以及通过作业训练数据分析、优化、模拟和绩效归因。
- QuantStart Beginner's Guide to Quantitative Trading: 量化交易流程可拆成策略识别、回测、执行系统和风险管理；其中数据质量、幸存者偏差、交易成本、滑点、最大回撤和 Sharpe Ratio 都是基础内容。
- QuantConnect Algorithm Framework: 一个可复用的量化系统通常拆成标的选择、Alpha 信号、组合构建、风险管理和执行模块。
- CFA Level I 相关课程结构: 专业投资学习通常覆盖伦理、数量方法、经济学、财务报表、公司金融、权益、固定收益、衍生品、另类投资和组合管理。这个项目不需要照搬 CFA，但可以借鉴其“工具 -> 资产 -> 组合”的顺序。

## 调研结论

1. 小白课程应该先讲“为什么投资”和“风险是什么”，再讲“股票是什么”。
2. 美股量化课程不能只讲策略，还要讲市场机制、数据口径、交易费用和执行限制。
3. 金融基础不需要变成完整 CFA 课程，但至少要覆盖收益、风险、复利、现值、资产类别、分散化和组合。
4. 股票基础不能只讲买卖，还要讲公司所有权、普通股/优先股、分红、拆股、市值、行业、指数、ETF、财报和估值入门。
5. 量化部分应该按研究流程组织：提出假设、获取数据、清洗数据、生成信号、构建策略、回测、评估、控制风险、模拟交易。
6. 回测必须较早讲清常见陷阱，包括未来函数、幸存者偏差、过拟合、样本内/样本外、交易成本和滑点。
7. 课程应该避免直接进入高频交易、复杂期权、杠杆、做空、深度学习或实盘自动交易。这些内容可以作为后续拓展，不应放进第一版核心路径。

## 建议课程路径

### 1. 投资与金融基础

目标：让学习者知道投资在解决什么问题，以及收益、风险和时间之间的关系。

建议内容：

- 储蓄、投资和投机的区别
- 复利、收益率、年化收益
- 风险、波动和亏损
- 利率、现金和无风险收益
- 投资目标、时间跨度和风险承受能力
- 资产类别：现金、债券、股票、基金和 ETF

### 2. 美股市场如何运转

目标：让学习者理解自己面对的是一个有规则、有参与者、有成本的市场。

建议内容：

- 交易所、券商、做市商和监管机构
- 美股交易时间和盘前盘后
- 买入、卖出、成交、盘口和流动性
- 市价单、限价单和常见订单概念
- 费用、税务和信息披露的基础意识

### 3. 股票基础

目标：让学习者知道股票代表什么，以及股票价格为什么会变。

建议内容：

- 股票代表公司所有权
- 普通股、优先股、分红和投票权
- 市值、行业、成长股、价值股、蓝筹股
- 指数、指数基金和 ETF
- 财报、盈利、估值和市场预期的入门概念

### 4. 风险、收益与组合

目标：把单只股票思维推进到组合思维。

建议内容：

- 简单收益率和对数收益率
- 波动率、回撤、最大回撤
- 相关性和分散化
- 基准收益和超额收益
- 风险调整收益
- 资产配置和再平衡

### 5. 公司、估值与基本面入门

目标：让学习者知道公司经营数据为什么会影响股票价格，并理解常见基本面指标。

建议内容：

- 收入、利润、现金流和资产负债表的最小必要概念
- 每股收益、营收增长、利润率和负债
- 市盈率、市净率和股息率
- 财报、公告和市场预期
- 为什么价值、质量和成长可以成为量化信号来源

### 6. 量化投资的思维方式

目标：从“凭感觉判断”切换到“用规则表达假设并验证”。

建议内容：

- 什么是量化投资
- 策略、信号、规则和参数
- Alpha、Beta、基准和风格暴露
- 动量、均值回归、价值、质量等基础想法
- 为什么简单策略也需要严格验证

### 7. Python 与数据基础

目标：让学习者能用最少的 Python 和 pandas 能力读取、整理和观察金融数据。

建议内容：

- Python、Notebook 和表格数据
- pandas / DataFrame 基础
- 时间序列、交易日和缺失值
- 价格、成交量、复权价格
- 分红、拆股和退市数据
- 数据质量和数据偏差

### 8. 策略信号入门

目标：让学习者写出第一批可解释的策略信号。

建议内容：

- 移动平均和趋势
- 收益率排序和动量
- 价格偏离和均值回归
- 成交量和波动率信号
- 信号检查和可解释性
- 信号方向、持有期和调仓频率

### 9. 回测与绩效评估

目标：让学习者知道回测能说明什么，不能说明什么。

建议内容：

- 回测的基本流程
- 样本内和样本外
- 交易成本、滑点和流动性
- 基准和买入持有比较
- 年化收益、波动率、Sharpe Ratio、最大回撤
- 未来函数、幸存者偏差和过拟合

### 10. 组合构建与风险控制

目标：把信号转成仓位，并控制单一股票和整体组合风险。

建议内容：

- 标的池选择
- 等权、按信号加权和简单风险加权
- 仓位大小和现金比例
- 止损、止盈和风险预算的基础概念
- 再平衡和换手率

### 11. 执行、模拟交易与工程化

目标：让学习者理解从回测到模拟交易之间还有工程和执行差距。

建议内容：

- 订单生命周期
- 回测成交和真实成交的差异
- 日志、参数、版本和可复现性
- 模拟交易和实盘前检查
- 为什么第一版不鼓励直接实盘自动交易

### 12. 综合项目

目标：把前面的概念串起来，完成一个小而完整的美股量化研究流程。

建议内容：

- 选择一个清晰的投资假设
- 准备数据
- 生成信号
- 构建组合
- 回测并评估
- 写出策略报告和风险说明

## 第一版建议边界

第一版核心课程建议只覆盖美股股票和 ETF，不把复杂期权、期货、外汇、加密货币、高频交易、杠杆和实盘自动化作为主线内容。

后续如果要扩展，可以单独做进阶模块，而不是塞进入门主线。
