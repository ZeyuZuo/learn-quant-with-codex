# 课程章节设计

这份文档只设计章节，不写正式课程内容。

设计依据来自 `docs/course-research.md` 中的调研资料：Investor.gov 和 FINRA 的投资者教育资料用于前置金融与市场基础；MIT OCW 金融课程用于风险、收益、组合和估值顺序；QuantStart 和 QuantConnect 用于量化研究流程、回测、风险管理和执行模块。

## 设计原则

- 先解决小白最容易缺失的金融常识，再进入股票和量化。
- 每个大章节都要承接前一章，不跳过必要概念。
- 每个小课只承担一个清晰学习目标。
- 每个大章节的小课数量按知识密度决定，不要求相同。
- 这里只设计课程序列，不写正式正文。
- 第一版主线只覆盖美股股票和 ETF，不深入期权、高频、杠杆和实盘自动交易。

## 1. 投资与金融基础

来源参考：Investor.gov、FINRA 的投资入门资料。

1. 为什么需要投资  
   目标：区分储蓄、投资和投机，理解投资要解决的问题。
2. 钱的时间价值与通胀  
   目标：理解今天的钱和未来的钱为什么不一样。
3. 收益率、复利和年化  
   目标：知道如何描述投资收益，并理解复利的影响。
4. 风险、波动和亏损  
   目标：建立“收益和风险一起看”的基本意识。
5. 利率、现金和无风险收益  
   目标：理解现金、短期国债和无风险收益为什么会影响投资比较。
6. 投资目标、期限和资产类别  
   目标：把投资选择和个人目标、时间跨度、资产类别联系起来。

## 2. 美股市场如何运转

来源参考：Investor.gov、FINRA 的市场与交易基础资料。

1. 市场里有哪些参与者  
   目标：认识公司、投资者、交易所、券商、做市商和监管机构。
2. 交易所、券商和账户  
   目标：理解普通投资者通过什么路径参与美股市场。
3. 报价、订单和成交  
   目标：理解买价、卖价、市价单、限价单和成交的基本关系。
4. 交易时间、流动性和成本  
   目标：知道交易不是免费的，价格和成交质量会受流动性影响。
5. 监管、公告和信息披露  
   目标：理解公开信息、公司公告和监管规则为什么重要。

## 3. 股票基础

来源参考：Investor.gov 的股票资料、MIT 金融课程中的权益资产基础。

1. 股票代表什么  
   目标：理解股票是公司所有权的一部分。
2. 普通股、优先股和分红  
   目标：认识常见股票类型、分红和股东权利。
3. 市值、行业和股票风格  
   目标：理解大盘股、小盘股、成长股、价值股等基础分类。
4. 指数、ETF 和基准  
   目标：理解指数和 ETF 的作用，并为后续基准比较做准备。
5. 拆股、财报和公司事件  
   目标：知道公司事件会影响价格数据和投资判断。

## 4. 风险、收益与组合

来源参考：MIT Finance Theory I、投资组合基础课程结构。

1. 简单收益率和对数收益率  
   目标：理解量化里常见的收益率表达方式。
2. 波动率、回撤和最大回撤  
   目标：用更具体的指标描述风险和亏损体验。
3. 相关性和分散化  
   目标：理解为什么多个资产组合起来可能降低风险。
4. 基准收益和超额收益  
   目标：学会把策略表现和可比较对象放在一起看。
5. 风险调整收益  
   目标：理解为什么同样收益下，波动和回撤更小的组合更有价值。
6. 资产配置和再平衡  
   目标：理解组合不是一次买入后永远不变。

## 5. 公司、估值与基本面入门

来源参考：MIT 金融课程中的估值和权益资产基础、CFA 入门框架中的财务报表与权益部分。

1. 财报的最小地图  
   目标：认识利润表、资产负债表和现金流量表各自回答的问题。
2. 收入、利润和利润率  
   目标：理解公司赚钱能力的基础指标。
3. 市盈率、市净率和股息率  
   目标：认识最常见的估值指标及其局限。
4. 增长、质量和负债  
   目标：理解基本面数据为什么可以形成价值、质量、成长信号。
5. 预期、公告和数据限制  
   目标：知道市场价格反映预期，基本面数据也会滞后和修正。

## 6. 量化投资的思维方式

来源参考：QuantStart 的量化交易流程、QuantConnect Algorithm Framework。

1. 从投资想法到研究假设  
   目标：把模糊想法改写成可以验证的问题。
2. 规则、信号和参数  
   目标：理解量化策略由可重复规则组成。
3. Alpha、Beta 和基准  
   目标：知道策略收益需要和市场暴露区分开。
4. 因子思维入门  
   目标：理解动量、价值、质量等因子为什么常见。
5. 研究记录和可复现性  
   目标：建立记录假设、参数和结果的习惯。

## 7. Python 与数据基础

来源参考：量化教程中的数据准备流程、QuantStart 对数据质量和偏差的强调。

1. 研究环境和 Notebook  
   目标：建立最小可用的量化学习环境。
2. Python 最小语法  
   目标：掌握变量、列表、字典、循环和函数这些后续必需语法。
3. pandas 和 DataFrame 基础  
   目标：理解表格数据的筛选、排序、列计算和缺失值处理。
4. 读取 CSV 和行情数据  
   目标：能把简单价格数据读进研究环境并检查字段。
5. 表格数据和时间序列  
   目标：理解价格数据为什么通常按日期组织。
6. 价格、成交量和复权  
   目标：知道原始价格和复权价格的区别。
7. 交易日、缺失值和公司行为  
   目标：理解金融数据不等于普通连续日历数据。
8. 画图和数据检查  
   目标：用简单图表和统计值发现异常数据。

## 8. 信号与简单策略

来源参考：量化入门教程中的趋势、动量、均值回归和因子信号。

1. 趋势和移动平均  
   目标：用价格序列构造最基础的趋势信号。
2. 动量和排序  
   目标：理解用过去表现给股票排序的基本方法。
3. 均值回归  
   目标：理解价格偏离和回归假设。
4. 基本面信号  
   目标：把估值、质量、成长数据转成简单信号。
5. 信号检查和可解释性  
   目标：检查信号分布、极端值和经济含义，避免只看结果。
6. 从信号到策略规则  
   目标：明确买入、卖出、持有期和调仓频率。

## 9. 回测与绩效评估

来源参考：QuantStart 对回测、交易成本、偏差和绩效指标的讨论。

1. 回测到底在模拟什么  
   目标：理解回测是按历史时间推进的规则检验。
2. 仓位、现金和调仓  
   目标：知道策略如何从信号变成持仓变化。
3. 交易成本、滑点和流动性  
   目标：理解真实交易约束会改变回测结果。
4. 基准和买入持有比较  
   目标：把策略和简单可投资替代方案进行比较。
5. 收益、波动、夏普和最大回撤  
   目标：掌握最基本的策略评价指标。
6. 样本内、样本外和时间切分  
   目标：理解为什么不能只在同一段历史上调参和评价。
7. 未来函数、过拟合和参数敏感性  
   目标：认识最常见的回测误区，并检查参数变化是否过度影响结果。

## 10. 组合构建与风险控制

来源参考：QuantConnect 的组合构建和风险管理模块、投资组合理论基础。

1. 标的池如何选择  
   目标：理解股票池会影响策略结果。
2. 仓位大小和权重  
   目标：认识等权、信号加权和简单风险加权。
3. 组合约束  
   目标：理解单票上限、行业暴露、现金比例等限制。
4. 换手率和再平衡  
   目标：知道调仓频率会影响成本和风险。
5. 风险规则和监控  
   目标：建立组合层面的风险控制意识。

## 11. 执行、模拟交易与工程化

来源参考：QuantStart 的执行系统、QuantConnect 的执行模块。

1. 订单生命周期  
   目标：理解从下单到成交之间发生了什么。
2. 回测、模拟交易和真实交易的差异  
   目标：知道回测表现不能直接等同于实盘表现。
3. 参数、日志和配置  
   目标：让研究结果可以复查和复现。
4. 实盘前检查清单  
   目标：明确第一版只做到模拟交易和风险检查，不鼓励直接实盘。

## 12. 综合项目

来源参考：成熟量化教程通常以完整研究项目收束，串联数据、信号、回测和报告。

1. 选择一个清晰假设  
   目标：提出一个能被数据验证的策略想法。
2. 准备数据集  
   目标：完成数据读取、清洗和检查。
3. 构建信号和组合  
   目标：把研究假设转成可运行策略。
4. 回测、评估和修正  
   目标：用指标和图表判断策略表现。
5. 写出研究报告  
   目标：说明假设、方法、结果、风险和下一步。
