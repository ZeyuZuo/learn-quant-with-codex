# Agent Notes

## 项目定位

这个仓库用于开发一个从零开始学习美股量化的中英双语教程网站。

目标用户是没有金融、股票或量化基础的小白。

## 已确认要求

- 做教程型学习网站。
- 参考 `https://learn.shareai.run/en/` 的页面风格和结构。
- 页面要简洁，重点放在课程内容上。
- 支持中文和英文。
- 课程设计需要循序渐进。
- 学完后，用户应对金融知识、股票概念和量化有比较全面的基础理解。

## 课程方向

用户已经明确要求基于外部教程和资料调研来扩充课程设计，不需要局限于最初的六个阶段。

除非用户后续修改，先按以下阶段设计课程：

1. 投资与金融基础
2. 美股市场如何运转
3. 股票基础
4. 风险、收益与组合
5. 公司、估值与基本面入门
6. 量化投资的思维方式
7. Python 与数据基础
8. 策略信号入门
9. 回测与绩效评估
10. 组合构建与风险控制
11. 执行、模拟交易与工程化
12. 综合项目

调研依据见 `docs/course-research.md`，小课程拆分见 `docs/course-outline.md`。这些是章节设计，不是最终课文内容。

## 写作要求

- 文档要像个人写的，语气自然。
- 公开文档要面向项目本身，不要直接写协作中的口语化原话。
- 不要把只适合内部沟通的措辞搬进 README 或页面文案。
- 不要随意扩展产品功能、课程范围或技术方案。
- 课程范围可以基于调研扩充，但要能说明为什么适合小白学习路径。
- 如果是产品功能或技术方案的新想法，先作为待确认事项告诉用户。

## 实现提醒

- 优先做课程型界面。
- 视觉设计要服务内容阅读。
- 从一开始就保留中英双语内容结构。
- 添加课程之外的能力前，先确认需求。
