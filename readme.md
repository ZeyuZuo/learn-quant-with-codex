# Learn Quant With Codex

一个从零开始学习美股量化的中英双语教程项目。

项目目标是做一个简洁、以课程为核心的学习网站，帮助初学者逐步理解金融基础、美股概念和量化投资。

## 当前范围

- 中英双语学习体验
- 面向小白的课程结构
- 从基础概念逐步过渡到量化投资
- 以课程内容为主的简洁页面

## 课程方向

当前课程路径：

1. 投资与金融基础
2. 美股市场如何运转
3. 股票基础
4. 风险、收益与组合
5. 公司、估值与基本面入门
6. 量化投资的思维方式
7. Python 与数据基础
8. 策略信号入门
9. 回测与绩效评估
10. 组合构建与风险控制
11. 执行、模拟交易与工程化
12. 综合项目

详细章节和实现方式还未最终确定。
